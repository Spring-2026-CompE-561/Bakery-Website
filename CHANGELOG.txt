FRONTEND SETUP INSTRUCTIONS:
----------------------------

To replicate the frontend environment on your machine:

1. Navigate to the frontend directory:
   cd frontend/

2. Install all dependencies:
   npm install
   
   This will automatically create node_modules/ with all required packages
   including React, React Router, and utilities.

3. Verify installation:
   npm list react react-router-dom
   
   Should show versions and confirm installation success.

4. Start development server:
   npm start
   
   Frontend will be available at http://localhost:3000/

5. Build for production:
   npm run build
   
   Creates optimized production build in build/ directory.

For Backend Issues:

- Check frontend/package.json for correct dependency versions
- Verify npm version: npm --version (should be recent)
- Clear npm cache: npm cache clean --force

For Database Integration:
- Ensure backend is running: uv run uvicorn src.app.main:app --reload
- Check DATABASE_URL in .env
- Verify SQLite file exists: bakery.db
